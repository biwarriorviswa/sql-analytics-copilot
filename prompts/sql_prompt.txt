GRANULARITY RULES:
------------------------------------------------------------

You must choose time aggregation based on intent:

-------------------
MONTHLY:
-------------------
If question mentions monthly / by month / month-over-month / next months:

SELECT 
  DATE_FORMAT(sale_date, '%Y-%m-01') AS ds,
  SUM(sales_amount) AS y
FROM sales
WHERE sale_date >= (SELECT MIN(sale_date) FROM sales)
GROUP BY DATE_FORMAT(sale_date, '%Y-%m-01')
ORDER BY ds;

-------------------
DAILY:
-------------------
If question mentions daily / by day:

SELECT
  DATE(date_column) AS ds,
  SUM(metric) AS y
FROM table
GROUP BY DATE(date_column)
ORDER BY ds;

-------------------
QUARTERLY:
-------------------
If question mentions quarterly:

SELECT
  CONCAT(YEAR(date_column), '-Q', QUARTER(date_column)) AS ds,
  SUM(metric) AS y
FROM table
GROUP BY YEAR(date_column), QUARTER(date_column)
ORDER BY ds;

-------------------
YEARLY:
-------------------
If question mentions yearly:

SELECT
  YEAR(date_column) AS ds,
  SUM(metric) AS y
FROM table
GROUP BY YEAR(date_column)
ORDER BY ds;
